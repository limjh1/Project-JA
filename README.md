# Project DS1 – Dev Steady 1 (Action RPG)

> Unreal Engine 5 C++ 기반의 **Gameplay Ability System (GAS)** 학습 및 확장 프로젝트  
> 강의: *Unreal Engine 5 C++ Advanced Action RPG (Vince Petrelli, Udemy)*  
> 엔진 버전: **Unreal Engine 5.5.4**

---

## Overview
**Project DS1** 은 C++과 GAS 구조를 실전 수준으로 익히기 위한 개인 프로젝트입니다.  
강의 기반으로 시작했으며, 이후 **커스텀 확장 및 구조 이해 중심의 포트폴리오**로 발전시킬 예정입니다.  

---

## Focus & Goals
- **Gameplay Ability System 완전 이해**  
  → Ability, Effect, Attribute, GameplayCue 간 데이터 흐름 체계화  
- **C++ 설계 패턴 습득**  
  → 재사용 가능한 모듈 구조 구현  
- **Action Combat 시스템 확장**  
  → Target Lock-on, Combo, Parry, EQS 기반 AI, Rage Mode 등 직접 구현 및 개선
